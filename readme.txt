Write the custom stock status with different colors for each WooCommerce simple product, to show in product details pages.

== Description ==

Custom stock status Plugin is an effective WooCommerce plugin which is used to Add extra stock option in WooCommerce. When you need to add an extra stock option in WooCommerce i.e. “request to buy” “ or request available” etc.  then simply you need to install this plugin.

This plugin is highly useful for WooCommerce platforms to inform their customers about the exact product status of certain products. You can change the default stock status and different colors for status text.

=== Keys Features and advantages of the extension: ===
* Custom Stock Status Message
* Custom Stock Status Message Color
* Show Status In Single Product Page

Please note: It only works on simple products, not on variable products.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wc-custom-stock-status` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the WooCommerce -> Settings -> Custom Stock(tab) screen to configure the stock status
1. You can see list of default stock status and text box to put your custom names
1. You can see the custom status text boxes in Simple product type screens

If you still need help. visit [WordPress codex](https://codex.wordpress.org/Managing_Plugins#Installing_Plugins)
